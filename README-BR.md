## 🌟 Sobre o Projeto Autumn

Autumn é um sistema de boletim escolar desenvolvido com o objetivo de facilitar o gerenciamento de notas, faltas e desempenho dos alunos. O sistema permite que professores, coordenadores e administradores tenham acesso rápido e seguro aos dados acadêmicos, promovendo transparência e eficiência no processo educacional.

---

## 💡 Exemplo de Uso

1. Um professor acessa o sistema para registrar as notas do 3º bimestre.
2. O coordenador gera um relatório consolidado das médias da turma.
3. Os pais consultam o boletim online para acompanhar o desempenho do aluno.

---

## 💻 Tecnologias Utilizadas

- **Front-End**: HTML, CSS e JavaScript para a interface do usuário.
- **Back-End**: PHP para processamento de dados.
- **Banco de Dados**: MySQL para armazenamento de informações acadêmicas.
- **Outras Ferramentas**: AJAX para comunicação assíncrona, Bootstrap/Bulma para design responsivo.

## 🚀 Pré-requisitos

Antes de começar, certifique-se de ter os seguintes itens configurados:

- **XAMPP instalado**: Necessário para rodar o servidor Apache e o banco de dados MySQL.
👉 [Baixe o XAMPP aqui !](https://www.apachefriends.org/download.html).

- **Navegador confiável**: Chrome, Firefox ou qualquer outro que permita explorar o projeto.

🛠️ **Instalação e Configuração**
Siga estas etapas simples para iniciar o Autumn:

1️⃣ **Configure o XAMPP**

- Instale e inicie o XAMPP. No painel de controle, ative os módulos Apache e MySQL.

2️⃣ **Configure o Banco de Dados**

- Abra o phpMyAdmin acessando: `http://localhost/phpmyadmin/`.
- Crie um banco de dados chamado `autumn` (ou qualquer nome preferido).
- Importe o arquivo `database/autumn.sql` localizado na pasta do projeto para criar as tabelas e dados iniciais.

3️⃣ **Obtenha o Blueprint**

- Você pode obter o projeto de duas maneiras:

- **Clonando o repositório**:

```bash
git clone https://github.com/seu-repositorio/autumn.git
```

- **Baixando o ZIP**:

- No repositório do projeto, clique em Code > Download ZIP. Extraia o conteúdo na pasta `htdocs` da instalação do XAMPP.

4️⃣ **Ajuste as Configurações**

- Localize o arquivo de configuração do banco de dados (geralmente em `config/db.php` ou similar). Atualize as credenciais para corresponder ao ambiente local.

5️⃣ **Execute o Projeto**

- Inicie o servidor no XAMPP. Abra o navegador e vá para: `http://localhost/autumn`
