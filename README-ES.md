## 🌟 Acerca del Proyecto Otoño

Autumn es un sistema de informe escolar diseñado para facilitar la gestión de las calificaciones, las ausencias y el rendimiento de los estudiantes. El sistema permite a docentes, coordinadores y administradores tener acceso rápido y seguro a los datos académicos, promoviendo la transparencia y eficiencia en el proceso educativo.

---

## 💡 Ejemplo de uso

1. Un docente accede al sistema para registrar las calificaciones del tercer semestre.
2. El coordinador genera un informe consolidado de los promedios de la clase.
3. Los padres consultan la tarjeta de calificaciones en línea para monitorear el desempeño del estudiante.

---

## 💻 Tecnologías utilizadas

- **Front-End**: HTML, CSS y JavaScript para la interfaz de usuario.
- **Back-End**: PHP para procesamiento de datos.
- **Base de datos**: MySQL para almacenar información académica.
- **Otras herramientas**: AJAX para comunicación asincrónica, Bootstrap/Bulma para diseño responsivo.

## 🚀 Requisitos previos

Antes de comenzar, asegúrese de tener configurados los siguientes elementos:

- **XAMPP instalado**: necesario para ejecutar el servidor Apache y la base de datos MySQL.
👉 [¡Descarga XAMPP aquí!](https://www.apachefriends.org/download.html).

- **Navegador de confianza**: Chrome, Firefox o cualquier otro que permita explorar el proyecto.

🛠️ **Instalación y configuración**
Siga estos sencillos pasos para comenzar con Autumn:

1️⃣ **Configurar XAMPP**

- Instalar e iniciar XAMPP. En el panel de control, habilite los módulos Apache y MySQL.

2️⃣ **Configurar la base de datos**

- Abra phpMyAdmin accediendo a: `http://localhost/phpmyadmin/`.
- Crea una base de datos llamada `otoño` (o cualquier nombre preferido).
- Importe el archivo `database/autumn.sql` ubicado en la carpeta del proyecto para crear las tablas y datos iniciales.

3️⃣ **Obtén el plano**

- Puedes obtener el proyecto de dos maneras:

- **Clonación del repositorio**:

```bash
git clone https://github.com/seu-repositorio/autumn.git
```

- **Descargando el ZIP**:

- En el repositorio del proyecto, haga clic en Código > Descargar ZIP. Extraiga el contenido en la carpeta `htdocs` de su instalación de XAMPP.

4️⃣ **Ajustar configuración**

- Localice el archivo de configuración de la base de datos (generalmente en `config/db.php` o similar). Actualice las credenciales para que coincidan con su entorno local.

5️⃣ **Ejecutar el proyecto**

- Iniciar el servidor en XAMPP. Abra su navegador y vaya a: `http://localhost/autumn`
